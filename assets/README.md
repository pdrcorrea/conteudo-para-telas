# Assets

Pasta de recursos estáticos do projeto ConteúdoParaTelas.

## Estrutura

```
assets/
├── logo/       # Logotipos PontoView (SVG, PNG)
└── images/     # Imagens das páginas (banners, screenshots, ícones)
```

## Convenções de nomenclatura

- Logos: `pontoview-logo.svg`, `pontoview-logo-white.svg`, `pontoview-favicon.png`
- Imagens de seção: `hero-bg.jpg`, `como-usar-step-1.png`, etc.
- Formatos preferidos: **SVG** para vetores/logos, **WebP/PNG** para fotos e screenshots
